Prototype Instructions

Navigating to journeys

This prototype is quite large and is used for a number of reasons, but primarily for iterating design, user research and communicating designs to developers.

The service overview diagram below shows how reports flow through the service.

Each user role has a journey in this prototype. Each journey is referenced at the top of the index page (1-4, where "4 Make Decision journey" includes Decision Makers and Business Support journeys).

Following these links will take you to the relevant part of the index page, which will reference one, two or maybe more versions of the journey.

Navigating versions

For example, the "Report incident or concern" journey has following features.

  • A. "UCD - IN PROGRESS"
    Section containing the latest designs. A mix of pages being design and those already released so that the new pages have context.

  • B. Link to the latest version of the journey
    Follow link to view the journey in order (Press "continue" mainly

  • C. Page index
    Expand details to link to an individual page.

  • D. Journey-level notes
    Expand details to display a summary of the work undertaken on this version of the journey. Latest versions will contain wireflow diagrams.

Page design notes

There are a number of features on each page which hold further information of page designs (which can be turned off by changing to Presentation mode on the index page).

  • E. Versions
    You are at the highlighted version. Click back to see the development of the design

  • F. States
    Different states of the page. Most often there is an error page showing warning messages for validation. There may also be alternative presentations based on different input to the same page.

  • G. Design notes
    Very brief notes from design/content to explain features where necessary

  • H. Hypothesis and insights
    Where appropriate, links to UR that drives design decisions

  • I. Journey level notes
    A repeat of the brief summary of design changes on this journey
