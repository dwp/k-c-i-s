Accessibility statement for Keep Customer Interactions Safe

This accessibility statement applies to the Keep Customer Interactions Safe (KCIS) service only.

Using this service

The Keep Customer Interactions Safe (KCIS) service is run by the Department for Work and Pensions (DWP). We want as many people as possible to be able to use this service. For example, this means you should be able to:

  • change colours, contrast levels and fonts
  • zoom in up to 300% without the text spilling off the screen
  • get from the start of the service to the end using just a keyboard
  • get from the start of the service to the end using speech recognition software
  • listen to the service using a screen reader (including the most recent versions of JAWS, NVDA and VoiceOver)

We’ve also made the text in the service as simple as possible to understand.

AbilityNet has advice on making your device easier to use if you have a disability.

How accessible this service is

We have designed this service to be fully accessible.

Feedback and contact information

If you have any difficulty using this service, email kcis.feedback@dwp.gov.uk.

Reporting accessibility problems with this service

We're always looking to improve the accessibility of this service. If you find any problems that are not listed on this page or you think we’re not meeting accessibility requirements, email kcis.feedback@dwp.gov.uk.

Enforcement procedure

The Equality and Human Rights Commission (EHRC) is responsible for enforcing the Public Sector Bodies (Websites and Mobile Applications) (No. 2) Accessibility Regulations 2018 (the ‘accessibility regulations’). If you’re not happy with how we respond to your complaint, contact the Equality Advisory and Support Service (EASS).

Technical information about this service’s accessibility

DWP is committed to making this service accessible, in accordance with the Public Sector Bodies (Websites and Mobile Applications) (No. 2) Accessibility Regulations 2018.

Compliance status

This website is fully compliant with the Web Content Accessibility Guidelines version 2.1 AA standard.

Preparation of this accessibility statement

This statement was prepared on 29 August 2023. It was last reviewed on 29 August 2023.

This service was last tested on 5 July 2023. The test was carried out by DWP, using a combination of manual and automated testing, including assistive technology.

The KCIS team originally submitted a sample of 20 pages to be tested by the DWP accessibility team. Their report shows that all of these pages are fully compliant with the WCAG version 2.1 standard.

Since the DWP accessibility team tested these sample pages, the KCIS project team have tested all new features in the service. We have now tested more than 80% of the pages in the service. We believe that all the pages we have tested are fully compliant with the WCAG version 2.1 standard.
